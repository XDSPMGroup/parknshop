DETAILS

Login and Modify personal information

  • Click the “Login” button, then show the following page.
  • Input seller’s account and password, click “Login” button then you will go to the homepage

Manage Cart

Before this case start, the user must login the system using an customer account.

  • Go to the home page of the commodity you selected.
  • nput the number of the commodity which you want to buy, and then click the “add to cart” button to add the record to the cart.
  • Click the “My Cart” button, it can be showed that the commodity is added to the My Cart.
  • Click the delete icon 1 to delete the commodity you select.
  • Click the delete icon 2 to delete all the commodities in the cart.

Buy Commodity

Before this case start, the user must login the system using an customer account.

  • Go to the home page of the customer.
  • Click the “My Cart” button.
  • Click “Checkout” button to pay.
  • After paying successfully, the cart is empty.

Manage order

  • Go to the home page of the customer.
  • Click the “Manage Order” button, enter the Manage Order Page.
  • You can see the information of the order and input the comment to the commodity you buy.

Add to favorite

Before this case start, the user must login the system using an customer account.

  • Go to the home page of the commodity you selected (refer to the scan commodity function in tourist model).
  • Click the “add to favorite” button to add commodities to customers’ favorites
  • The “add to favorite” button will turn to “already to favorite”, and the commodity can be seen in your favorite.
  • Click “My Favorite” link to scan the favorites.

Manage favorite

Before this case start, the user must login the system using an customer account.

  • Go to the home page of the customer.
  • Click “My Favorite” link to scan the favorites.
  • (add to cart):Click “Add to Cart”, add the commodity to cart.
  • then add successfully, show the page:
  • (delete collections):Click “Del” button to delete the record from your favorite.

Scan Shopping History

Before this case start, the user must login the system using an customer account.

  • Go to the home page, Click the “BuyHistory” button.
  • Enter the Buys History page, scan the buy history.
  • Then you can scan year, month, day buy history.(refer to the store owner’s check sales history function).

Scan top 10 commodities and shops

Before this case start, the user must login the system using an customer account.

  • Go to the home page of the customer.
  • (scan top10 commodities):Click the “Top 10 Commodities” link to scan top 10 commodities
  • Click the first commodity, enter the commodity page.
  • (scan top10 shops):Click the “Top 10 Shops” link to scan top 10 shops.
